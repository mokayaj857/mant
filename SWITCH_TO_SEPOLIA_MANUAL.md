# How to Switch to Sepolia Testnet Manually

If you're getting the "Unrecognized network" error, follow these steps to switch to Sepolia Testnet manually:

## Option 1: Switch via MetaMask UI (Easiest)

1. **Open MetaMask** (click the extension icon)
2. **Click the network dropdown** at the top (it might say "Ethereum Mainnet" or another network)
3. **Scroll down** and look for "Sepolia" in the list
4. **Click "Sepolia"** to switch to it
5. **Refresh the page** and try minting again

## Option 2: Add Sepolia Network Manually

If Sepolia doesn't appear in your network list:

1. **Open MetaMask**
2. **Click the network dropdown**
3. **Click "Add Network"** or "Add a network manually"
4. **Enter these details**:
   - **Network Name**: `Sepolia`
   - **RPC URL**: `https://rpc.sepolia.org`
   - **Chain ID**: `11155111`
   - **Currency Symbol**: `ETH`
   - **Block Explorer URL**: `https://sepolia.etherscan.io`
5. **Click "Save"**
6. **Switch to Sepolia** from the network dropdown
7. **Refresh the page** and try minting again

## Option 3: Use MetaMask's Test Networks

1. **Open MetaMask Settings** (click the three dots menu)
2. **Go to "Advanced"**
3. **Enable "Show test networks"** (toggle it on)
4. **Go back** and click the network dropdown
5. **You should now see Sepolia** in the list
6. **Click Sepolia** to switch
7. **Refresh the page** and try minting again

## Verify You're on Sepolia

After switching, you should see:
- Network name shows "Sepolia" in MetaMask
- Chain ID: 11155111
- The page should detect the correct network

## Still Having Issues?

If you're still getting errors after switching:
1. **Refresh the browser page** completely (Ctrl+Shift+R or Cmd+Shift+R)
2. **Disconnect and reconnect** your wallet
3. **Check the browser console** for any error messages
4. **Make sure your MetaMask is updated** to the latest version
